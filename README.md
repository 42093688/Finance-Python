## finpy
-------------

纯python实现的金融计算库，参考quantlib

## 依赖
-------------

~~~
enum34 (python2)
numpy
~~~

## 安装
-------------

根据使用的python版本不同，例如Python2:

~~~
pip install -r requirements/python2.txt
python setpy.py install
~~~
